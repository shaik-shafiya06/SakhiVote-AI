# SakhiVote AI

## Problem Statement
Creating an assistant that helps users understand the election process, timelines, and steps in an interactive and easy-to-follow way. Especially targeted for first-time voters, rural users, and low-literacy users.

## Solution
SakhiVote AI is a voice-first, multilingual, highly accessible web assistant. It requires absolutely no reading—everything is guided by voice. It adapts dynamically to all device languages, offers inclusive modes (Guided vs Quick), and simulates the voting process seamlessly in the browser.

## Features
- **Voice-First Interaction:** Powered by Web Speech API. Speaks and listens in the user's local language.
- **Inclusive Onboarding:** Language selection followed by simple Mode Selection (Guided vs Quick).
- **Interactive Voting Simulator:** Practice using an Electronic Voting Machine (EVM).
- **Step-by-Step Guidance:** Explains registration, documents, and the voting process.
- **Accessibility Engine:** Light/Dark modes, Large Text, High Contrast, and Replay Voice capabilities.
- **Offline & Lightweight:** Entirely runs in the browser using HTML/CSS/JS with zero external API dependencies and <10MB footprint.

## Tech Stack
- HTML5
- CSS3 (Custom Variables, Animations)
- Vanilla JavaScript (ES6+)
- Web Speech API (SpeechSynthesis & SpeechRecognition)

## How to Run
1. Clone or download this repository.
2. Open `index.html` in any modern web browser (Chrome/Edge recommended for best Speech API support).
3. Ensure you have microphone permissions enabled when prompted.

## Social Impact
Empowers marginalized, illiterate, and first-time voters by breaking down complex democratic processes into a friendly, story-like audio experience. Promotes voter inclusion and confidence.
